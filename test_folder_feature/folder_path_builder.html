🗂️ フォルダパス構築システム

ステップ1: Rootフォルダパスを入力

ベースとなるルートフォルダのフルパスを入力してください

例: C:\Users\YourName\Documents\Projects

ステップ2: 個別フォルダ名を入力

個別の作品フォルダ名を直接入力してください（確実・シンプル・アラートなし）

フォルダ名が入力されていません
例: MyGame2025, ProjectABC2024, TestFolder など
✅ 利点: アラートなし、全ブラウザ対応、確実な動作
📝 使い方: エクスプローラーでフォルダ名を確認してから入力

ステップ3: フルパスを構築・コピー

Root + 個別フォルダの組み合わせフルパス

フルパスがまだ構築されていません

ステップ4: 動作確認

コピーしたパスを貼り付けて確認
