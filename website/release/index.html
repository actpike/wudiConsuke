『ウディこん助』は、WOLF RPGエディターコンテスト（通称：ウディコン）のファンが作成した非公式のChrome拡張機能です。
作品管理を目的とし、プレイ済み作品のチェック・感想管理、新作や更新の検出までバッチリ対応。 あなたのウディコン体験をもっと快適に。

🔗 Chrome Web Store ✨ 機能を見る

主な機能

📊

評価・感想管理

ウディコン公式の6カテゴリ評価システムに対応。プレイした作品の評価と感想を一元管理。各評価の平均値表示で、あなたの評価傾向も把握できます。

📝

感想メモ機能

作品ごとに詳細な感想を記録。文字数カウント機能付きで、コメント投稿時の参考にも最適。大切な作品の印象を忘れずに保存できます。

🔔

自動監視・通知

ウディコンサイトを開いた時に自動で新規作品・更新をチェック。バージョン更新や新規投稿を見逃すことなく、デスクトップ通知でお知らせします。

💾

データ管理

評価・感想データのエクスポート・インポート機能。大切なデータを安全に保護します。

導入方法

Chrome Web Storeで公開中！簡単インストールまたは手動インストールをお選びください。

Chrome Web Store から簡単インストール

公式のChrome Web Storeからワンクリックでインストールできます。

🏪 Chrome Web Store で入手

⚠️ この拡張機能はPC版Chromeブラウザでのみ動作します

インストール手順

  1. 上記ボタンをクリックしてChrome Web Storeを開く
  2. 「Chromeに追加」ボタンをクリック
  3. 確認ダイアログで「拡張機能を追加」をクリック
  4. ブラウザのツールバーにアイコンが表示されたら完了！

手動インストール（zip版）

開発版や最新版を試したい場合は、こちらから手動でインストールできます。

📦 ウディこん助 v0.0.6 をダウンロード

⚠️ この拡張機能はPC版Chromeブラウザでのみ動作します

導入手順

1
Chrome拡張機能管理画面を開く

    【事前準備】上記のボタン[ ダウンロード ]より、
    zipを任意のフォルダへ解凍

  1. Chromeブラウザで chrome://extensions/ にアクセス
  2. 右上の『開発者モード』を有効化
  3. 押下『パッケージ化されていない拡張機能を読み込む』
  4. 解凍したzipのフォルダを選択し、押下[ フォルダーの選択 ]
2
インストール完了

ウディこん助がインストールされ、ブラウザのツールバーにアイコンが表示されます。クリックして利用開始！

仕様と責任について

本ツール（拡張機能）は「あったら便利かも！」と個人の趣味で制作した非公式のツールです。
利益の獲得や個人情報の収集を目的としたものではありません。

入力されたデータは原則として外部に送信されません。
すべてのデータはローカル（ブラウザのキャッシュ領域）に保存されます。

下記の操作を行った場合、保存されたデータは削除されます：

  • 拡張機能の削除
  • オプション画面の「全データ削除」ボタンを押す
  • 各作品ページで「削除」ボタンを押す
  • ブラウザのキャッシュをクリアする（※環境によります）

なお、以下の操作ではデータは削除されません：

  • 拡張機能のON/OFFの切り替え
  • 拡張フォルダの再配置・上書き（zip再インストールなど）

入力した評価・感想などを大切に保存しておきたい場合は、設定画面から「データエクスポート」を行っておくことをおすすめします。

本ツールの使用により発生したいかなるトラブル・損害についても、制作者は一切の責任を負いません。
ご了承ください。
